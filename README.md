# FashionVault

Frontend Angular application for managing a digital wardrobe, creating outfits, and planning clothing choices based on weather, activities, and daily context.

---

## Features

### Dashboard

**Features**
- Wardrobe statistics (items, style distribution, trends)
- Recently worn carousel
- Outfit calendar overview

## Tech Stack

- Angular
- TypeScript
- HTML
- CSS
- RxJS (services for state management)
- Mock JSON data

---

## Architecture

- Component-based structure
- Reusable UI components (cards, carousels, modals)
- Feature-based page organization
- Centralized services for state management
- DRY and encapsulated logic

---

## Project Structure

- **components** — reusable UI components  
- **pages** — application pages  
- **services** — business logic and state management  
- **data** — mock data with local images  
- **models** — TypeScript interfaces  

---

## Future Improvements

- Backend integration (.NET Web API)
- SQL Server database
- Authentication system (user accounts and wardrobe sync)
- AI-based outfit recommendation system (style + context-aware suggestions)
- Image processing for outfit analysis (detect clothing items from uploaded images)
- Weather API integration for real-time outfit planning
- Mobile version
